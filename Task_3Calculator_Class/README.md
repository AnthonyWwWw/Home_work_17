## TASK_3

# Calculator Class

Цей клас надає методи для виконання базових математичних операцій.

## Використання

1. Створіть новий екземпляр класу Calculator.
2. Використовуйте методи для виконання математичних операцій.

## Методи

### `add(num1, num2)`

Додає два числа разом і повертає результат.

### `subtract(num1, num2)`

Віднімає одне число від іншого і повертає результат.

### `multiply(num1, num2)`

Множить два числа разом і повертає результат.

### `divide(num1, num2)`

Ділить одне число на інше і повертає результат. Перевіряється, чи друге число не дорівнює нулю.

## Приклад використання

```javascript
const calc = new Calculator();

console.log(calc.add(5, 3)); // 8
console.log(calc.subtract(10, 4)); // 6
console.log(calc.multiply(3, 6)); // 18
console.log(calc.divide(8, 2)); // 4
